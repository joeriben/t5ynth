T5ynth

This installer will set up T5ynth on the selected Mac volume:

  • T5ynth.app in /Applications/
  • Factory presets in /Library/Application Support/T5ynth/presets/
  • Model storage in /Library/Application Support/T5ynth/models/

The installer creates the shared T5ynth support folder and installs the T5ynth app on that volume.

After installation, launch T5ynth and use the Settings panel to download a model.

T5ynth is open-source software (GPLv3).
