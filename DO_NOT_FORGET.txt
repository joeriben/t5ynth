macOS: After downloading, open Terminal and run this command.

  How to open Terminal:
    - Press Cmd+Space (Spotlight), type "Terminal", press Enter
    - Or: Finder → Applications → Utilities → Terminal

  Then paste this command and press Enter:

    xattr -cr /Applications/T5ynth.app

(Adjust the path if you placed T5ynth.app elsewhere.)

What this does:
  xattr  = extended attributes tool (built into macOS)
  -c     = clear all extended attributes
  -r     = recursively (the .app bundle contains many files)

Why this is needed:
  macOS adds a "com.apple.quarantine" attribute to every file downloaded
  from the internet. When you try to open a quarantined app, macOS checks
  whether it is signed by a registered Apple Developer and notarized.
  T5ynth is open-source software (GPLv3) and is not signed with an Apple
  Developer certificate ($99/year). macOS will therefore refuse to open it
  and display "T5ynth.app is damaged" — it is not actually damaged.
  Removing the quarantine attribute tells macOS to skip this check.
