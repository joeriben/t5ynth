# UCDCAE AI Lab Wordmark CD Info

Context: UCDCAE AI Lab affiliation in T5ynth public material and manual header.

The UCDCAE AI Lab wordmark is a single-line uppercase wordmark for affiliation
and attribution. Do not split `UCDCAE` and `AI LAB` into separate lines, and do
not recolor it with T5ynth project accent colors.

## Text

`UCDCAE AI LAB`

## Typography

- T5ynth manual desktop font size: `0.95rem`
- T5ynth manual mobile font size: `0.78rem`
- Font weight: `700`
- Line height: `1`
- Letter spacing desktop: `0.14em`
- Letter spacing mobile: `0.1em`
- White-space: `nowrap`
- Font family: inherit the T5ynth manual sans-serif stack
